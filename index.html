La ligne de commande

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : 

mkdir NomDuFichier

Pour créer un fichier, je dois utiliser la commande : 

touch NomDuFichier

Pour se déplacer dans un dossier, je dois utiliser la commande : 

cd NomDuDossier

Pour déplacer un dossier, je dois utiliser la commande : 

mv NomDuDossier

Pour connaitre le chemin courant, je dois utiliser la commande : 

pwd

Pour se mettre en superutilisateur, je dois utiliserr la commande : 

sudo -s

Pour sortir du mode superutilisateur, je dois utiliser la commande : 

exit

Pour supprimer un fichier, je dois utiliser la commande : 

 rm NomDuFichier

Pour afficher le contenu d'un dossier, je dois utiliser la commande : 

ls

Pour trouver un paquet, je dois utiliser la commande : 

sudo apt search NomDuPaquet

Pour lire un fichier en lecture seule, je dois utiliser la commande : 

cat NomDuFichier
